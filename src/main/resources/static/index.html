API 사용 설명서 by 이영욱

ARC 를 사용해 API를 확인할 수 있습니다!

기능 Http Method URL @RequestBody (JSON) Return
1. 전체 게시글 조회 GET http://52.78.132.235/api/postings List <PostingResponseDto>
2. 게시글 작성 POST http://52.78.132.235/api/postings { "title" : "제목", "writer" : "작성자", "content" : "내용"} PostingResponseDto
3. 게시글 조회 GET http://52.78.132.235/api/postings/{id} PostingResponseDto(CommentResponseDto)
4. 게시글 수정 PUT http://52.78.132.235/api/postings/{id} { "title" : "수정할 제목", "writer" : "수정할 작성자", "content" : "수정할 내용"} posting_id
5. 게시글 삭제 DELETE http://52.78.132.235/api/postings/{id} posting_id
6. 댓글 작성 POST http://52.78.132.235/api/postings/{id}/comments { "comment" : "내용"} @ResponseBody String (성공 혹은 실패 메시지)
7. 댓글 수정 PUT http://52.78.132.235/api/comments/{id} { "comment" : "수정할 내용"} @ResponseBody String (성공 혹은 실패 메시지)
8. 댓글 삭제 DELETE http://52.78.132.235/api/comments/{id} comment_id
